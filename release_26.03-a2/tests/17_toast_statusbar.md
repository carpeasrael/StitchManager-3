# TC-17: Toast Notifications & Status Bar

- **Steps:** Save metadata or export file
- **Expected:** Green toast with ✓ icon, auto-dismiss after 4s
- **Status:** PENDING

## TC-17-02: Error Toast
- **Steps:** Trigger an error (e.g., export to invalid path)

- **Status:** PENDING

## TC-17-03: Info Toast
- **Steps:** Trigger watcher auto-import (add file to watched dir)

- **Status:** PENDING
