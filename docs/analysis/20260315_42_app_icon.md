# Analysis: Use design/icon_stitichManger.png as app icon

**Date:** 2026-03-15

## Problem description

Replace the current app icons with the new design icon from `design/icon_stitichManger.png` (1024x1024 RGB PNG).

## Affected components

- `src-tauri/icons/` — all icon files (PNG sizes, .icns, .ico)
- Source: `design/icon_stitichManger.png`

## Root cause / rationale

New app icon design needs to be deployed across all platform icon formats.

## Proposed approach

1. Generate all required PNG sizes from the 1024x1024 source using `sips` (macOS built-in)
2. Generate `.icns` using `iconutil` (macOS built-in)
3. Generate `.ico` using `sips` + manual assembly or `png2ico`
4. Replace all files in `src-tauri/icons/`

Required sizes:
- Standard: 32, 64, 128, 256 (128@2x), icon.png (512)
- Windows Square logos: 30, 44, 71, 89, 107, 142, 150, 284, 310
- StoreLogo: 50x50
- macOS: icon.icns (16,32,128,256,512 + @2x variants)
- Windows: icon.ico (16,24,32,48,64,128,256)
